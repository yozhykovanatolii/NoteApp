## How to set up a project
### Steps:
1. Make sure you have Flutter installed.
2. Clone this repository:

   ```
   git clone https://github.com/yozhykovanatolii/NoteApp.git
   ```
4. Navigate to the project directory:

   ```
   cd NoteApp
   ```
6. Run the app:

   ```
   flutter run
   ```
